    MP3fm

    "MP3fm" stands for "MP3 Folder Making app" which AUTOMATICALLY Pack Songs into folders according to user choice from TITLE/ARTIST/ALBUM/YEAR/DURATION/COMMENT.

    I believe that for Music Lovers its a big problem to keep songs organized into the folders, so mp3fm is a simple solution to that problem. Just run the app, select the folder which contain the songs and it will give you the options of Packing the Songs into folders based on the properties like Album(Movie)/Artist/Year/Comments/Title/Duration, Unpacking songs from already existing folders and Updating the Properties of songs present in that folder.

    Features

    • PACK : Move songs into folders based on property choosen by user from Album(Movie)/Artist/Year/Comments/Title/Duration.

    • UNPACK : Move songs into choosen folder from already existing folders inside it.

    • UPDATE : Update the Properties of all the songs(ID3 metadata) present in the choosen folder.

    • GENERATE LOG : LOG file is created after every operation which helps in mapping new song location.

    Please refer to MP3fm Documentation documentation for additional details.

    Instructions to Follow:

    • Install mp3fm Tool using:

    $ pip install mp3fm

    • Using Terminal just run the App & follow the instructions:

    $ mp3fm

    and then just follow the instructions displayed on screen.
