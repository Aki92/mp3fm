.. _usage:

****************
Usage & Download
****************

Just run the App and follow the instructions shown in it:


Unix
####

.. code-block:: sh

		$ mp3fm

:download:`Download mp3fm.tar.gz <downloads/mp3fm-1.0.1.tar.gz>`


Windows
#######

Open 

.. code-block:: sh

		mp3fm.exe
		
:download:`Download mp3fm.zip <downloads/mp3fm-1.0.1.zip>`
